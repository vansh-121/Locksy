# ⌨️ Keyboard Shortcuts

> **✅ READY TO USE**: Keyboard shortcuts are **pre-configured** and work out-of-the-box! Using number keys to avoid conflicts.

## 🎯 Default Shortcuts

The extension comes with these pre-assigned shortcuts:
- `Alt+Shift+9` - Lock Current Tab
- `Alt+Shift+0` - Open Domain Lock Manager
- `Alt+Shift+8` - Lock All Tabs
- `Alt+Shift+7` - Toggle Stealth Mode

**Why number keys?** They rarely conflict with other extensions or browser shortcuts!

## ⚙️ Customize (Optional)

Want different shortcuts? You can change them:

1. **Navigate to Shortcuts Page:**
   - Chrome: `chrome://extensions/shortcuts`
   - Edge: `edge://extensions/shortcuts`
   - Brave: `brave://extensions/shortcuts`

2. **Find "Locksy"** in the extension list

3. **Change Any Shortcut:**
   - Click the pencil icon (✏️) next to the command
   - Press your desired key combination
   - Save and test!

## 📋 Available Commands

### 🔒 Lock Current Tab
- **Default**: `Alt + Shift + 9`
- **Action**: Instantly locks the currently active tab
- **Alternatives**: `Alt + Shift + L`, `Alt + Shift + Q`, `Ctrl + Shift + Period`

### 🌐 Open Domain Lock Manager
- **Default**: `Alt + Shift + 0`
- **Action**: Opens the Domain Lock Manager window
- **Alternatives**: `Alt + Shift + D`, `Alt + Shift + M`

### 🔐 Lock All Tabs
- **Default**: `Alt + Shift + 8`
- **Action**: Locks all tabs in the current window at once
- **Alternatives**: `Alt + Shift + A`, `Alt + Shift + K`

### 🕵️ Toggle Stealth Mode
- **Default**: `Alt + Shift + 7`
- **Action**: Toggles Stealth Mode — hides badge counter and suppresses all lock-state notifications
- **Note**: Enabling is always silent; disabling shows a brief confirmation notification
- **Also available**: Right-click context menu and popup Stealth Mode toggle section

## 💡 Conflict Resolution

**If shortcuts don't work:**

1. **Check for conflicts**: Another extension might be using the same keys
2. **Try alternatives**: Use different combinations like:
   - `Alt + Shift + [Letter]` (safest option)
   - `Ctrl + Shift + [Special key]` (Period, Comma, etc.)
   - `Alt + Shift + [Number]` (0-9)

3. **Avoid these combinations** (used by browsers):
   - ❌ `Ctrl + D` (Bookmark)
   - ❌ `Ctrl + T` (New tab)
   - ❌ `Ctrl + W` (Close tab)
   - ❌ `Ctrl + Shift + A` (Search tabs in Chrome)
   - ❌ `Ctrl + Shift + D` (Bookmark all in Chrome)
   - ❌ Most `Ctrl + [Letter]` combinations

4. **Best practices**:
   - ✅ Use `Alt + Shift` for least conflicts
   - ✅ Choose less common letters (Q, X, Z)
   - ✅ Use special keys (Period, Comma, Brackets)
   - ✅ Try number keys if letters are taken

## 🎯 Usage Tips

1. **First Time Setup**: Set your master password before using lock shortcuts
2. **Quick Toggle**: Assign a shortcut to quickly activate/deactivate extension
3. **Bulk Protection**: Use lock-all when you need to step away quickly
4. **System Pages**: Shortcuts automatically skip pages that cannot be locked
5. **Test After Setup**: Try each shortcut to ensure it works

## Troubleshooting

**Shortcut not working?**
- Check if extension is activated (use popup toggle switch)
- Ensure master password is set
- Verify the tab is not a system page
- Check for conflicts with other extensions

**Want different keys?**
- Navigate to browser shortcuts settings
- Customize any shortcut to your preference
- Avoid conflicts with browser defaults

## Advanced Usage

### Quick Lock Workflow
1. Browse to sensitive tab
2. Press `Alt+Shift+L` to lock
3. Move to another task
4. Unlock with password when needed

### Bulk Security Mode
1. Open multiple sensitive tabs
2. Press `Alt+Shift+A` to lock all
3. Step away safely
4. Return and unlock as needed

### Domain Lock Workflow
1. Press `Alt+Shift+D` to open manager
2. Add domain patterns
3. All matching tabs auto-lock
4. New tabs auto-lock too
